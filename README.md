# üè• HMPV Virus Tracker & Medical System

A comprehensive full-stack medical portal built with **Django (Python)**. It features a Patient Dashboard for test requests and vaccination records, and a Secure Admin Panel for verification and approval.

## üåü Key Features
* **Secure Authentication:** Separate Login flows for Patients and Admins.
* **Test Requests:** Patients submit symptoms + Aadhaar ID.
* **WhatsApp OTP:** Integrated Twilio API for phone verification.
* **Admin Dashboard:** Admins review IDs and approve/reject tests.
* **Auto-Certificate:** System generates a PDF Certificate and Emails it automatically upon approval.
* **Vaccination Tracker:** Digital record of vaccination history.
